This repository contains materials for the ProWiT (Prodapt Women in Tech) initiative — a 2-day AI Internship / Nano Bootcamp program focused on building real-world Agentic AI applications.

Program Name: ProWiT – Prodapt Women in Tech

Duration: 16 Hours
Format: Hands-on · Project-Based · Industry-Focused
Conducted By: Prodapt

📌 Held on: 17th and 18th April 2026

🚀 About the Program

This intensive 2-day program is designed to take participants from fundamentals of Generative AI to building fully functional AI agents.

The curriculum covers all topics outlined in the official agenda, including:

Generative AI fundamentals
Large Language Models (LLMs) & APIs
Prompt Engineering
Vector Databases & RAG pipelines
AI Agents & Agentic Workflows
Tool Usage, Function Calling & ReAct
LangGraph Framework
Capstone Project: Telecom Domain AI Agent

All sessions include hands-on labs and real-world implementations, ensuring practical learning.

🏗️ Learning Outcome

By the end of this program, participants will be able to:

Understand and work with modern LLMs and APIs
Design effective prompts and AI workflows
Build RAG-based systems using vector databases
Develop AI agents with tool usage and memory
Implement stateful agent systems using LangGraph
Deliver a production-style AI solution (capstone project)
